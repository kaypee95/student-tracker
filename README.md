# Student Tracker v1.0

A command-line student tracker demonstrating CRUD operations with JavaScript arrays and objects.

## 🎯 Purpose

This is a learning project for the PASSO API Development Course. It demonstrates:
- Working with arrays of objects
- CRUD operations (Create, Read, Update, Delete)
- Organizing code into modules
- Data manipulation in JavaScript

## 📁 Project Structure

student-tracker/
├── data.js   # Initial student data
├── crud.js   # CRUD functions
├── app.js    # Main application
└── README.md # This file

## 🚀 How to Run
```bash
# Navigate to project folder
cd student-tracker

# Run the application
node app.js
```

## 📚 Features

**Create:** Add new students with auto-incrementing IDs

**Read:** Get all students, search by name, grade, or ID

**Update:** Modify any student property with automatic timestamp

**Delete:** Remove students by ID

**Utilities:** Count students, list all unique grades

## 💡 What I Learned
- Working with JavaScript modules (require/module.exports)
- CRUD operation patterns
- Array methods (map, filter, find, etc.)
- Object manipulation
- Console formatting
